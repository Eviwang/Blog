[blog.jerry-wang.cn](http://blog.jerry-wang.cn) Source Code.

```yml
  # 创建文章
  hexo new <title>

  # 创建草稿
  hexo new draft <title>

  # 草稿发布到文章
  hexo publish draft <title>

  # 创建新页面
  hexo new page "about"

  # 发布到GitHub
  hexo deploy -g

  # 启动Server
  hexo server -p 5000

  # 生成静态文章
  hexo -g

  # 清理缓存（hexo分类不正确时，可能是缓存的原因）
  hexo clean
```
